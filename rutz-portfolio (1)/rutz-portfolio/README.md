# rutz — macOS Portfolio

Personal portfolio website styled as a macOS desktop.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Framework: Vite
4. Build command: `npm run build`
5. Output: `dist`
